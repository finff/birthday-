# Interactive Birthday Website

A responsive static birthday surprise website built from the instructions in `interactive_birthday_website_plan.md` and `task(1).md`.

## How to Open

Open `index.html` in a browser. No build step is required.

## Personalize Before Sharing

- Replace the photo placeholders in `index.html` with real images in `assets/images/`.
- Add your song as `assets/music/our-song.mp3`.
- Replace `Sayang`, `Your name`, memory dates, timeline dates, and captions with personal details.
- Keep the audio muted or absent until the opening button is clicked; the current code already follows that rule.

## Included Features

- Opening gift animation and birthday countdown.
- Music controls with play, pause, mute, and volume.
- Hero section, animated gallery modal, love cards, and typing messages.
- Catch My Love game with score, timer, golden hearts, broken hearts, high score, and reward.
- Memory matching game, birthday letter, cake interaction, timeline, hidden secrets, confetti, fireworks, and final surprise.
- Responsive layout, keyboard support, focus states, semantic sections, alt/ARIA labels, and reduced-motion support.
