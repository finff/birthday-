# Interactive Birthday Website for My Girlfriend

## Project Overview

Create a beautiful, romantic, and interactive birthday website for my girlfriend.

Her birthday is **27 June 2003**.

The website should feel sweet, playful, personal, and memorable. It should include animated photos, small interactive surprises, a mini-game, birthday wishes, and smooth transitions.

---

## Main Theme

### Primary Color

- Baby Blue: `#BDE0FE`

### Supporting Colors

Use these colors together with baby blue:

- Soft Lavender: `#CDB4DB`
- Blush Pink: `#FFC8DD`
- Soft White: `#FFF9FB`
- Dark Navy Blue for text: `#2D3A5A`
- Light Gold for small highlights: `#F6D58A`

### Design Style

The website should look:

- Soft and romantic
- Cute but not childish
- Modern and clean
- Dreamy and elegant
- Mobile-friendly
- Smooth and interactive

Use rounded cards, soft shadows, floating hearts, stars, sparkles, clouds, flowers, and subtle gradients.

Suggested background gradient:

```css
background: linear-gradient(
  135deg,
  #bde0fe 0%,
  #e2d1f9 45%,
  #ffc8dd 100%
);
```

## 5. Mini-Game: Catch the Hearts

Create a small game called:

> Catch My Love

### Game Instructions

- Hearts appear randomly around the game area.
- She must click or tap the hearts before they disappear.
- Each heart gives one point.
- The game lasts for 20 to 30 seconds.
- Special golden hearts give three points.
- Avoid clicking broken hearts because they reduce the score.

### Score Messages

#### Score 0–5

> You may have missed some hearts, but you will never miss a place in mine. 💙

#### Score 6–12

> You caught so much love! But I still have more love waiting for you.

#### Score 13–20

> Amazing! You caught almost all my love.

#### High Score

> You caught every piece of my heart. It has always belonged to you anyway. 💖

### Reward After Completing the Game

After the game ends, display:

- Confetti
- A hidden photo
- A secret birthday message
- A button to unlock the next surprise

Button text:

> Claim Your Birthday Reward

## Responsive Design Requirements

The website must work properly on:

- Mobile phones
- Tablets
- Laptops
- Desktop screens

Important requirements:

- Large and easy-to-tap buttons
- Photos should resize automatically
- Text should remain readable
- The mini-game must support touch controls
- No horizontal scrolling
- Animations should be lighter on smaller devices

---

## Accessibility

Include:

- Alt text for every image
- Clear button labels
- Good text contrast
- Keyboard-accessible buttons
- A mute button for music
- A reduced-motion option
- Readable font sizes
